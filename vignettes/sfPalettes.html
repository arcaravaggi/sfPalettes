sfPalette Usage

Anthony Caravaggi

2017-02-19

To display a colour swatch, simply call the SF_palette function and pass a palette name. Palette names can be found in the internal mov dataframe. All colours are displayed by default.

library(sfPalettes)
data(mov)
mov
##    Colour.code                             Movie.title
## 1         ANH                               A New Hope
## 2        AOTC                     Attack of the Clones
## 3         ESSM  Eternal Sunshine of the Spotless Mind 
## 4         HER                                      Her
## 5          JP                            Jurassic Park
## 6         RGO                                Rogue One
## 7        ROTJ                       Return of the Jedi
## 8        ROTS                      Revenge of the Sith
## 9        TESB                  The Empire Strikes Back
## 10        TFA                        The Force Awakens
## 11         TM                              The Martian
## 12        TPM                       The Phantom Menace

The Empire Strikes Back

SF_palette("TESB")

You can also specify the number of colours displayed by using the n = parameter.

Her

SF_palette("HER", n = 4)

Palettes can be interopolated by extending n = beyond the number of available colours and using type = continuous.

Eternal Sunshine of the Spotless Mind

SF_palette("ESSM", n = 24, type = "continuous")

You can also use SF_palettes to colour your plots, as below:

Scatterplot with the Jurassic Park palette

library("ggplot2")
## Warning: package 'ggplot2' was built under R version 3.3.2
ggplot(iris, aes(Sepal.Length, Sepal.Width, color = Species)) +
  theme_bw() +
  geom_point(size = 3) +
  scale_color_manual(values = SF_palette("JP")) +
  labs(y = "Sepal width", x = "Sepal length") +
  theme(legend.text = element_text(face = "italic"))

Stacked barplot using The Martian

qplot(factor(cyl), data = mtcars, geom = "bar", fill=factor(vs)) +
  scale_fill_manual(values = SF_palette("TM"))

A mountain created with Her

SF_colours_1 <- SF_palette("HER", 21, type = "continuous")
image(volcano, col = SF_colours_1, las = 1)

Heat map using Eternal Sunshine of the Spotless Mind

data(heatmap)  

SF_colours_2 <- SF_palette("ESSM", 100, type = "continuous")

ggplot(heatmap, aes(x = X2, y = X1, fill = value)) + 
  geom_tile() + 
  scale_fill_gradientn(colours = SF_colours_2) + 
  scale_x_discrete(expand = c(0, 0)) +
  scale_y_discrete(expand = c(0, 0)) + 
  coord_equal()
